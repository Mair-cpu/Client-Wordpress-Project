## 🚀 Project Summary
**Acid Money** is a premium WordPress-based web application tailored for a gaming community. It features a high-contrast dark-mode aesthetic designed to maximize user retention and engagement.

### Core Features:
* **Dynamic Leaderboards**: Real-time visualization of user rankings and wagering activity.
* **Incentive Systems**: Streamlined "Claim Bonus" workflows and VIP perk integrations.
* **Integrated Media Gallery**: A curated grid layout for community videos and updates.
* **Responsive Architecture**: Fully optimized for a seamless experience on mobile, tablet, and desktop.
